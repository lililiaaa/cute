# 可爱多好物分享
一个个人分享好物，音乐等的网站页面
使用html,css实现静态页面，使用h5,css3实现动态效果，利用js,jquery实现前端页面交互。使用响应式布局，有移动端，PC端，手机端三种布局方式；
